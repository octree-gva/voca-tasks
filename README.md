# Decidim::VocacityGemTasks

vocacity_gem_tasks.

## Usage

VocacityGemTasks will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-vocacity_gem_tasks"
```

And then execute:

```bash
bundle
```
